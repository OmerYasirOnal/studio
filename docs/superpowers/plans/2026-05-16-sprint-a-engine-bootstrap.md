# Sprint A — Engine Bootstrap Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Scaffold `games/brave-bunny/app/` with Vite + React + Three.js + React Three Fiber, get an empty 3D scene running in browser, wrap with Capacitor 7 for iOS, and produce TestFlight Build #6 via the existing fastlane pipeline.

**Architecture:** Three streams, each ending with a clean commit:
- **Stream 1 (scaffold)** — `app/` Vite + TS + R3F project boots in browser at `localhost:5173` showing a rotating colored cube. Vitest + Playwright wired. CI green via `bb-web-test`, `bb-lint`, `bb-e2e`.
- **Stream 2 (iOS shell)** — `npx cap add ios` generates `games/brave-bunny/ios/`. Capacitor config carries bundle ID + name. Xcode workspace builds clean (unsigned, local). CI green via `bb-ios-build`.
- **Stream 3 (TestFlight)** — Fastlane `Fastfile` updated to point at the Capacitor workspace. `fastlane beta` signs + uploads. Build appears on TestFlight as Build #6.

**Tech Stack:** Vite 6+, React 19, TypeScript 5.6+, three.js r170+, @react-three/fiber 9, @react-three/drei, @react-three/rapier, miniplex, zustand, Vitest, Playwright, @capacitor/core 7, @capacitor/ios 7, ESLint 9, Prettier 3. Existing: fastlane, match, App Store Connect API key.

**Spec source:** `docs/superpowers/specs/2026-05-16-engine-pivot-design.md` §3-7.

```tsx
import { useFrame } from '@react-three/fiber';
import { useRef } from 'react';
import { type Mesh } from 'three';

export default function RotatingCube() {
  const ref = useRef<Mesh>(null);

  useFrame((_, delta) => {
    if (!ref.current) return;
    ref.current.rotation.x += delta * 0.5;
    ref.current.rotation.y += delta * 0.8;
  });

  return (
    <mesh ref={ref} position={[0, 0.5, 0]}>
      <boxGeometry args={[1, 1, 1]} />
      <meshStandardMaterial color="#ff6f3c" />
    </mesh>
  );
}
```
