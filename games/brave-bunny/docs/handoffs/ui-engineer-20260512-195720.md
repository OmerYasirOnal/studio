## The 8-property binding interface (verbatim)

namespace Brave.UI.Bindings
{
    public interface IRunRuntimeState
    {
        float CurrentHP { get; }
        float MaxHP { get; }
        float CurrentXP { get; }
        float XPToNextLevel { get; }
        int Level { get; }

Implementations must be allocation-free per read. `RunHudController.State` is a settable property; when `null` the controller falls back to a private `RunHudStubRuntime` instance so the HUD renders placeholders in editor.

## Allocation note

Per-frame `Render()` uses `string.Format` (interpolated strings) for HP/timer/wave/level text. This is the Wave-5 baseline. Zero-alloc upgrade path (Span<char> + `int.TryFormat`) is tracked for a Wave-6 polish dispatch — gate it on profiler evidence that UI text allocations show up in 60-fps frame budget.
