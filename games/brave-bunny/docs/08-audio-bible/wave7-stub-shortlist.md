## Round-robin authoring (asset-curator ‚Üí sound-designer)

Per `02-sfx-spec.md` ¬ß Round-robin enforcement:
1. Pull the **single source clip** from the CC0 source.
2. In Audacity, generate variants by:
   - Micro-pitch ¬±50 cents (different per variant).
   - Envelope tweak: ¬±10 % attack / ¬±5 % decay.
3. Export `<slug>_01.ogg` ‚Ä¶ `<slug>_NN.ogg` at 96 kbps mono.
4. Record CC-BY attributions in `LICENSES.md` if any source carries CC-BY (Pixabay tracks where licence is mixed).
