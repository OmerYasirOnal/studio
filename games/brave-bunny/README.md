# Brave Bunny

Built on the Studio framework. See the repo-root `README.md` for what Studio is and how it works. Read `GAME.md` in this directory for this game's concept and config. Game documentation lives under `docs/`.

| Where | What |
|---|---|
| `GAME.md` | Game config (genre, platforms, target devices, etc.) |
| `CLAUDE.md` | Game-specific Claude Code rules — extends the root `CLAUDE.md` |
| `docs/` | All design docs, tech specs, ADRs, handoffs |
| `assets-raw/` | CC0 source assets — see `LICENSES.md` |
| `data/balance/` | Source-of-truth JSON tuning |
| `unity/` | Unity 6 LTS project (empty until Phase 5) |
| `tools/ci/` | Fastlane + GitHub Actions for builds |
| `logs/` | Per-game agent logs |
